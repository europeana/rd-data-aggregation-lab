About the prototype

This prototype is a work in progress. It aims to support the work at Europena on the normalisation of date values in EDM.
Currently, not all planned functionalities are implemented. The extraction of approximate, unknown and uncertain dates is only partially implemented.

This tool provides the following:

  • Test date normalisation on a record in Europeana
  • Normalise a date value

Statistics of the Europeana dataset

The tool has been applied to the whole Europeana dataset. Some statistics regarding the matched date patterns may be accessed in the following links:

  • Occurrences across the whole dataset
  • Occurrences by collection
  • Occurrence by class
