About the prototype

This prototype is a work in progress. It aims to support the work at Europena on the normalization of date values in EDM.

This tool provides the following:

  • Test date normalization on a record in Europeana
  • Normalize a date value

For using the tool, follow the link for the type of solution to be tested.

The tool has been applied to the whole Europeana dataset. Some statistics regarding the matched date patterns may be accessed in the following links:

  • Occurrence of date patterns across the whole dataset
  • Occurrence of date patterns by collection
