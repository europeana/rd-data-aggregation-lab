About the prototype

This prototype is a work in progress. It aims to support data providers of Europeana in checking the compliance of their implementation of solutions for delivering data to Europeana.

This tool supports checking of the following:

  • Wikidata entity about a cultural heritage object
  • Schema.org included in a webpage (as used for Internet search engines)
  • IIIF manifest with structured metadata in Schema.org or EDM (seeAlso references)
  • sitemap.xml and robots.txt of a website
  • Evaluate data quality of a record at Europeana

For using the tool, follow the link for the type of solution to be tested.
